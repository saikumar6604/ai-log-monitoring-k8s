# 🚀 AI-Based Log Monitoring & Auto-Healing System

## 📌 Overview

This project demonstrates a system that monitors application logs and detects anomalies using Machine Learning.

## 🏗️ Architecture

* Dockerized Flask Application
* Kubernetes Deployment (Pods, Services)
* AI-based Anomaly Detection (Isolation Forest)
* Monitoring using Prometheus & Grafana
* CI/CD using GitHub Actions

## 🔄 Workflow

1. Application generates logs
2. Logs are analyzed using AI model
3. Anomalies detected
4. Kubernetes auto-heals (restarts pods)
5. Monitoring dashboards track system health

## 🛠️ Tech Stack

Docker | Kubernetes | Python | Prometheus | Grafana | GitHub Actions

## 🚀 How to Run

1. Build Docker image
2. Deploy to Kubernetes
3. Access via NodePort
